
## Correr Localmente

Clona el proyecto

```bash
git clone https://github.com/RenzoDotJava/delosi-test
```

Dirigete al directorio

```bash
cd delosi-test
```

Instala las dependencias

```bash
npm install
```

Corre el proyecto

```bash
npm run start
```
